## 1. 项目概述

### 1.1 项目背景

建设一个个人品牌介绍网站，作为个人在互联网上的"数字名片"。网站承载个人信息展示、技能标签、作品集入口等功能，是个人品牌建设的核心阵地。

### 1.2 产品目标

| 目标 | 衡量标准 |
|------|----------|
| **信息传达** | 访客 3 秒内知道"这是谁、做什么" |
| **品牌印象** | 访客离开后能记住 1-2 个个人标签 |
| **行动转化** | 提供明确的联系方式或社交入口 |
| **可扩展** | 未来可无缝新增作品集、博客等页面 |

### 1.3 目标用户

| 用户类型 | 来意 | 核心关注点 |
|----------|------|-----------|
| **HR/猎头** | 评估候选人 | 技能标签、工作经历、项目经验 |
| **同行/潜在合作者** | 寻找合作机会 | 技术栈、作品、联系方式 |
| **随机访客** | 通过社交链接跳转而来 | 快速了解、留下印象 |

## 2. 信息架构

### 2.1 页面结构（V1 单页）

```
┌─────────────────────────────────┐
│  Nav（固定顶部、毛玻璃）          │
│  [Logo/姓名]  [关于] [技能] [联系] │
├─────────────────────────────────┤
│  Hero（全屏首屏）                 │
│  ┌───────────────────────────┐  │
│  │  头像占位（圆形）           │  │
│  │  姓名（大标题、衬线体）      │  │
│  │  一句话身份标签              │  │
│  │  简短介绍                   │  │
│  └───────────────────────────┘  │
├─────────────────────────────────┤
│  Section 2: 关于我（浅底）       │
│  ┌───────────────────────────┐  │
│  │  详细个人简介段落            │  │
│  │  关键数字/亮点（3 列卡片）    │  │
│  └───────────────────────────┘  │
├─────────────────────────────────┤
│  深色面板: 金句/个人哲学          │
├─────────────────────────────────┤
│  Section 3: 技能标签（浅底交替）  │
│  ┌───────────────────────────┐  │
│  │  标签云/标签卡片网格         │  │
│  │  分类：技术 / 领域 / 软技能   │  │
│  └───────────────────────────┘  │
├─────────────────────────────────┤
│  Section 4: 联系方式 CTA        │
│  ┌───────────────────────────┐  │
│  │  社交链接按钮                │  │
│  │  邮箱 / GitHub / etc.      │  │
│  └───────────────────────────┘  │
├─────────────────────────────────┤
│  Footer（版权、Powered by）      │
└─────────────────────────────────┘
```

### 2.2 多页扩展预留（V2+）

```
首页(index.html)  ─┬── 作品集(portfolio/index.html)
                   ├── 博客(blog/index.html)
                   └── 关于(about/index.html) — 或合并到首页
```

Nav 栏预留作品集和博客的导航位，V1 阶段以灰色/不可点击方式展示或隐藏。

## 3. 功能需求

### 3.1 V1 功能清单

| ID | 功能 | 说明 | 优先级 |
|----|------|------|--------|
| F01 | **Hero 首屏** | 头像占位 + 姓名 + 身份标签 + 短介绍 | P0 |
| F02 | **关于我 Section** | 详细个人简介，2-3 段落 | P0 |
| F03 | **技能标签** | 标签云或网格卡片形式，含分类 | P0 |
| F04 | **金句面板** | 深色背景 + 个人哲学/座右铭 | P1 |
| F05 | **联系方式 CTA** | 社交链接按钮（GitHub/邮箱等） | P0 |
| F06 | **导航栏** | 固定顶部 + 毛玻璃 + 锚点跳转 + 滚动阴影 | P0 |
| F07 | **滚动动效** | Scroll Reveal（IntersectionObserver + stagger延迟） | P1 |
| F08 | **响应式** | 900px / 600px 双断点适配 | P0 |
| F09 | **Footer** | 版权信息 | P1 |

### 3.2 V2 计划功能

| ID | 功能 | 说明 |
|----|------|------|
| F10 | **作品集页** | 项目卡片网格，每个项目含截图/描述/链接 |
| F11 | **博客页** | 文章列表，按时间倒序 |
| F12 | **多页导航** | Nav 链接跳转到子页面 |

### 3.3 交互定义

| 交互 | 触发条件 | 行为 |
|------|----------|------|
| **导航滚动效果** | `window.scrollY > 40px` | Nav 添加 `scrolled` class（阴影出现） |
| **锚点跳转** | 点击 Nav 链接 | `scroll-behavior: smooth` 滚动到对应 Section |
| **Scroll Reveal** | 元素进入视口 12% | 添加 `visible` class，触发 opacity + translateY 过渡 |
| **Stagger 延迟** | 同 Section 内多卡片 | `.reveal-d1` ~ `.reveal-d5`，间隔 0.1s |
| **CTA 按钮悬停** | `:hover` | `translateY(-2px)` + `box-shadow` 扩展 |

## 4. 非功能需求

| 类型 | 要求 |
|------|------|
| **性能** | 纯静态页面，首屏 < 1s；Google Fonts 用 `preconnect` 预热 |
| **无障碍** | 尊重 `prefers-reduced-motion`；语义化 HTML；图片有 alt |
| **SEO** | `<title>` + `<meta description>` + 语义化标题层级 |
| **可维护性** | CSS 用自定义属性集中管理；个人信息与页面结构分离（数据文档） |
| **浏览器兼容** | 现代浏览器（Chrome/Firefox/Safari/Edge 最近 2 个大版本） |

### 5.1 配色（硬约束）

| 角色 | 色值 | 用途 |
|------|------|------|
| 蓝 | `#2B7FD8` | 主色调（~60%）、标题、链接、CTA 按钮 |
| 黄 | `#F4D758` | 强调（~30%）、高亮标记、装饰、选中文本 |
| 红 | `#E84A5F` | 点缀（~10%）、特别标记 |
| 奶油底 | `#fefcf6` | 主背景 |
| 深奶油 | `#faf6eb` | 交替 Section 背景 |
| 墨色 | `#1A1A2E` | 正文文字 |
| 深色面板 | `#151821` | 金句/引用区域背景 |

### 5.2 字体

| 用途 | 字体 |
|------|------|
| 中文标题 | `Noto Serif SC` (900) |
| 中文正文 | `Noto Sans SC` (400/500/700) |
| 英文装饰 | `Fraunces` (italic) |
| 代码/标签 | `Fira Code` |
| 手写/注释 | `Caveat` |

### 5.3 通用禁止

- ❌ 纯黑 `#000` 或纯白 `#fff`
- ❌ 蓝紫渐变 / glassmorphism / neon
- ❌ bounce/elastic 动画
- ❌ HTML 默认样式（默认 blockquote、无样式 ul/ol/table）
- ❌ 所有 Section 居中布局千篇一律
- ❌ 看起来像"AI 生成的通用模板"
